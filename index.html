Protótipo sem backend para demonstração acadêmica

Métrica do Projeto

  • Onde aparece: dashboard principal.
  • Como é apresentada: número total diário.
  • Como é calculada: contagem das ocorrências com data igual à data atual.
  • De onde vêm os dados: dados simulados e registros feitos no protótipo.
  • Período analisado: últimas 24 horas do dia atual.
  • Fonte: registros simulados iniciais + registros locais cadastrados pelo usuário.
